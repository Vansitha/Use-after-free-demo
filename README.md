## Use After Free Demo Program

Step 1 - Compiling the program

`gcc uaf_demo.c -o uaf_demo`
`gcc uaf_demo_patched.c -o uaf_demo_patched.c`

Step 2 - Run the program

`./uaf_demo`
`./uaf_demo_patched`
